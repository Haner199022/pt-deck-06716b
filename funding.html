04
Profiles

三档场景画像

A
VC Seed · ¥200w

谁适合：想全力一搏 + 不介意股权稀释 15% + 想加速到 break-even。

✓ Pros

  • M3 招第二个工程师，迭代速度 2–3×
  • 营销预算 ¥40w 能跑赛事 + KOL 矩阵
  • ¥20w 储备让 M9–M12 不被现金流压死
  • 创始人可发 ¥1.5w 月薪，财务压力小

✕ Cons

  • 出让 15%——Y3 base case 每年少分 ~¥5w
  • 投资人施压 monthly metrics，影响产品节奏
  • VC 看 ARR 增长不看利润，可能逼烧钱
  • 需要董事会 / 决策流程
B
Friends & Family · ¥50w · RECOMMENDED

谁适合：手里 ¥10–20w 自有 + 能拉 ¥30–40w 亲友 + 不想被 KPI 绑架。

✓ Pros

  • 100% 控股，所有未来分红归你
  • 决策完全自主，无月度 board update
  • 18 月足够走到 M3 收费验证
  • 失败成本可控，亲友轮通常软债务化处理

✕ Cons

  • M3–M6 你一个人扛产品 + 营销 + 客服
  • 亲友钱是隐性债务，失败后人际关系复杂
  • 无法快速招人，市场窗口可能被竞品抢
  • 创始人 ¥0 月薪，要靠自有现金生活
C
Lean · ¥10w

谁适合：现金紧张 + 想"花最少试一下" + 有副业能补贴生活。

✓ Pros

  • 投入小，失败成本 ≈ 一辆电动车
  • 10–12 月强制 break-even 压力反而能聚焦核心
  • 100% 控股，0 股权稀释
  • 验证失败的话不影响生活

✕ Cons

  • 营销极其紧张，只能靠创始人 IP + 0 成本社群
  • 1 个 Bug 或 1 次 KOL 失败可能吃掉 1 月预算
  • M9 没收入回正基本就死了（无 buffer）
  • 长期 = "兼职 SaaS"，难全心投入

⚠ 关键瓶颈

最大的不确定不是触达池有多大，是 Free → Paid 转化率。

如果只有 15%（vs 假设 30%），M12 付费数只剩 140，36 月 ARR 跌到 ¥15w——直接砍掉一半。

这就是为什么 MVP 上线后 M3 是关键节点：首批 30 教练真实付费数据会校准这个核心参数。

敏感性 · 哪个杠杆最猛

3 个核心变量 ±1 单位对 36 月 ARR 的影响：

变量 基线 改善后 36 月 ARR 变化
月流失率 Churn ↓ 6% 5% +18%
加权 ARPU ↑ ¥90 ¥100 +11%
CAC ↓ ¥300 ¥250 LTV/CAC 4.5× → 5.4×
结论

降 churn 是最高 ROI 杠杆——好产品 + 教练社群运营 + 续费提醒 = 留存。比拼命拉新更划算 18%。这是产品和运营该把 80% 精力投入的方向。

06
Returns

回报预期

钱花出去之后，你（创始人）和投资人各自能拿什么？这是创业前必须算清楚的账。三个维度看清楚：成功路径月入 × 失败概率 × 投资人侧 EV。

创始人月度到手 · base case 成功路径

假设成功跑到 BP §10 水位场景：M3 起付费 30 教练 → Y3 break-even → Y5 站稳 → Y10 接近 SAM 顶。

时间节点 A · 15% 稀释 ¥200w 融资 B · 5% 稀释 ¥50w 亲友 C · 0% 稀释 ¥10w 自有
Y1 · 内测期 ¥1.5w 工资 / 亏 ¥0 / 自费 ¥0 / 自费
Y2 · 增长期 ¥1.5w/月 ¥0.5–1w/月 ¥0.5w/月
Y3 · break-even ¥3.6w/月 ¥3.9w/月 ¥4w/月
Y5 · 站稳 ¥9–12w/月 ¥11–15w/月 ¥11–16w/月
Y10 · 顶峰（SAM 30%） ¥40–80w/月 ¥50–95w/月 ¥50–100w/月
观察

三档差距 Y3 不明显（10–15%），但 Y5–Y10 复利后差距放大——Tier B/C 在 Y10 比 Tier A 多分 20–25%。15% 稀释看似不痛，10 年累计代价 ≈ ¥800w–1500w。

失败概率 · 风险加权后的真实期望

但只看成功路径不够——每档失败率不同。资金少 / 团队薄 / 营销弱 → 失败概率上升。

tier 成功率 失败率 Y3 风险加权月入
A · ¥200w 融资 ~50% ~50% ¥1.8w/月
B · ¥50w 亲友 ~40% ~60% ¥1.56w/月
C · ¥10w 极简 ~25% ~75% ¥1w/月
⚠ 这数字看着低

Y3 风险加权 ¥1–1.8w/月 看似不诱人——但这是"50–75% 概率拿 0"的 EV 平均。如果你**成功**那一半，月入 ¥3.6–4w；如果失败，月入 ¥0 + 投入清零。创业从来不是 EV 游戏，是"我赌我属于成功那一半"的信念游戏。

投资人回报 · 仅 Tier A 有投资人

如果走 Tier A 融资路线，投资人投 ¥200w 拿 15% 股权。VC 视角的 5 种退出场景：

退出场景 概率 投资人回款 MOIC
① 完全失败 / 资金烧完 ~50% ¥0 0×
② 僵尸公司 · dividend 回 ~20% ¥150–200w 0.75–1×
③ 被场馆 SaaS 普通收购 ~20% ¥360–720w 1.8–3.6×
④ 战略溢价收购 ~8% ¥1,200–2,400w 6–12×
⑤ IPO / 大额并购 ~2% ¥2,400–4,000w 12–20×
概率加权 EV 100% ~¥352w 1.76×
⚠ VC 视角的诚实判断

EV(MOIC) ≈ 1.76× → 5 年期 IRR ≈ 12%

  • 顶级早期基金 IRR 基准 25–35% → ❌ 远不及
  • 中位数早期基金 12–15% → 🟢 基本匹配
  • 这意味着：顶级 VC 不太可能下注，但中型基金 / 战略投资人会感兴趣

融资策略推荐

基于以上 EV 分析，Tier A 的真正 fit 投资人不是"经典 VC"，而是：

  • 健身行业战略投资人——安踏战投 / 蕉下 / Keep 战投部 / LVMH 体育投资，看协同价值不纯看 IRR
  • 个人天使——健身公众号大号 / 退休的 Keep / BodyPark 高管，单笔 ¥30–50w，更灵活
  • 地方政府引导基金——如果有"全民健身"产业方向匹配的基金

不推荐找：顶级人民币基金（红杉、高瓴等）—— 他们 IRR 要求 30%+，这个标对他们而言"小而精但回报曲线不够性感"。
